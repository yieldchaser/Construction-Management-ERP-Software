Company MOM vs Project MOM Feature Company-Level MOM Project-Level MOM Where Left sidebar, MOM Inside a project, MOM tab Scope All projects or no project That project only Extra column Project column shown No project column needed Best used for Cross-project meetings, company-wide reviews, board meetings Project-specific site meetings, design reviews Accessing Company MOM Click MOM in the left sidebar. The MOM List Column What It Shows Name Meeting name and date Attendee Number of attendees Project Project the meeting was associated with (blank if not project-specific) Notes Summary of what was discussed Filters Filter Options Search Search by meeting name Project Show only meetings linked to a specific project Date Filter All, Today, Last Week, Last Month, This Week, This Month, or Custom Attendee Show meetings attended by a specific person Creating a New MOM Click New MOM + in the top right. Field Required What to Enter Date Yes Auto-filled with today&#8217;s date Meeting Name Yes A descriptive name (e.g. &#8220;Monthly Management Review&#8221;, &#8220;Cross-Project Design Meeting&#8221;) Select Project No Leave blank for a company-wide meeting, or select a project to associate it Attendee No Search and add attendees from the company team Notes No Record agenda points, decisions, and action items Attach Media No Upload photos, documents, or supporting files Click Save . Tips Use company MOM for management-level reviews. Weekly or monthly reviews involving multiple projects should be logged here rather than inside one specific project. Leave Project blank for general meetings. Board meetings, staff training, or all-hands discussions that are not tied to any project should be saved without selecting a project.
