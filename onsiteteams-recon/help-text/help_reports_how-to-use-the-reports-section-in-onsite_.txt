Before You Start You need at minimum a role with Report access enabled. Admins and Senior Managers can see all reports by default. Some reports (Salary, Payroll) require Payroll data to be set up first. Others (Materials, PO) require data to be recorded in the respective modules. How to Open Reports Click Reports in the left sidebar. The Reports page shows all 16 categories at a glance. Each category displays its individual report names. How to Open and Filter a Report Each report has two icons: Icon What It Does Settings (gear) icon Opens the report in full view with all filters available. Use this when you want to apply date ranges, project filters, or party filters before downloading. Download icon Downloads the report immediately with default filters, without opening the full view. Use this for quick exports when no custom filtering is needed. Common filters available inside most reports: Filter What It Does Date Range Limit the report to a specific period Project Show data from one project or all projects Party Filter by a specific vendor, client, or subcontractor Status Filter by Paid, Unpaid, or Partially Paid where applicable After applying filters, click Download PDF or Download Excel inside the report view. Use the Download icon when you want the full report quickly with default filters. Most reports default to the last 30 days. Always check the date range filter when the report opens to ensure it covers the period you need. The Party Ledger report is the fastest way to get a full transaction history for any vendor or client. Filter by party name and set the date range to cover the full project duration. The PnL Report and Project Financial Summary are the best starting points for a management review meeting. Both give a quick revenue vs expense overview across all projects. The Unbilled Item Report in Materials and Inventory identifies GRNs with no matching purchase invoice. Run this weekly to keep accounts payable accurate. The Cost Code Wise Expense Analysis is the most useful report for overhead tracking when expenses are recorded with consistent cost codes. Library reports (Material Library, Party Library, etc.) are useful for auditing master data â€” checking for duplicates or inconsistencies before a bulk upload.
