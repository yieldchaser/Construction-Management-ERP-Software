# ONSITETEAMS COMPLETE API MAP
# Base: https://api.onsiteteams.in/apis/v3
# Auth headers: Authorization (Bearer JWT), Project-Company-Id, Project-Id, Version-Code: 171

## NAMESPACES DISCOVERED
- /list/          — main data listing
- /stats/         — statistics/summaries  
- /chart/         — chart data
- /dashboard/     — dashboard endpoints
- /subcon/        — subcontractor module
- /payroll/       — payroll module
- /transaction/   — transaction module
- /sales-expense/ — sales & expense module
- /payment-in-out/— payment tracking
- /unbilled-material/ — unbilled materials
- /count/         — count endpoints
